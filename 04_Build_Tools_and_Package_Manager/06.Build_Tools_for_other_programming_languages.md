# Build Tools for other programming languages

* Java: maven | gradle
* JavaScript: npm | yarn | webpack
* Python: pip
* C/C++: conan
* C#: NuGet
* Golang: dep
* Ruby: RubyGems

## Pattern in all these tools:
1. dependency file: package.json, pom.xml, build.gradle
2. repository for dependencies
3. command line tool
4. ackage managers
